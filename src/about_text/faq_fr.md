# Foire aux questions (FAQ)

Le Guide de l'utilisateur We Care et les vidéos d'instruction sont de bonnes ressources pour répondre à de nombreuses questions courantes concernant l'utilisation et les fonctionnalités de l'outil. D'autres questions fréquentes sont traitées ci-dessous :

#### **D'où proviennent les données par défaut de l'outil ?**

L'intention est que les pays trouvent et utilisent leurs propres données, et c'est ce que nous recommandons comme meilleure pratique. Les paramètres par défaut de l'outil sont uniquement un point de départ et peuvent être des estimations utiles pour les indicateurs où les données quantitatives sont rares. Les données par défaut comprennent des estimations régionales (Annexes A et B), ainsi que des données HHFA spécifiques aux pays, lorsqu'elles sont disponibles. Pour plus d'informations sur les sources de données, consultez la section sources de données de ce document.

#### **Comment puis-je mettre à jour les paramètres de mon pays ?**

L'utilisation de l'outil nécessite un ensemble de chiffres pour le scénario de référence et les scénarios d'intervention. Vous pouvez mettre à jour les paramètres spécifiques à votre pays directement dans la plateforme, ou en téléchargeant/chargeant le fichier CSV des paramètres comme indiqué dans la section guide de l'utilisateur de ce document.

#### **Comment puis-je enregistrer mes sources ?**

L'outil fournit un moyen d'entrer des notes et annotations, comme description de source de données. Vous pouvez enregistrer vos sources pour chaque indicateur dans la case « source » associée (voir la section guide de l'utilisateur pour plus de détails). Une fois ces informations saisies, elles seront téléchargées dans le fichier CSV des paramètres, ainsi que fournies comme contexte pour l'assistant IA. Ces sources servent uniquement à garder une trace des données et ne sont pas utilisées dans les calculs.

#### **Comment puis-je exporter mes sources ?**

Vous pouvez exporter les sources via le fichier CSV, ou demander à l'assistant IA de créer un tableau de sources ou une liste de citations pour vous, une fois que vous avez saisi les sources dans la case « source » associée pour chaque indicateur.

#### **Puis-je collaborer sur un projet avec plusieurs membres de l'équipe en même temps ?**

Vous pouvez collaborer avec les membres de l'équipe lorsque vous travaillez sur des projets, cependant la plateforme enregistrera les mises à jour et modifications uniquement dans votre navigateur (autrement dit, uniquement dans la version sur laquelle vous travaillez). Vous pouvez télécharger des fichiers de paramètres à partager entre les membres de l'équipe ou pour créer une sauvegarde, et partager des rapports et visualisations en les exportant depuis la plateforme (vers Word ou sous forme d'images PNG).

#### **D'où viennent les définitions dans l'onglet « Définitions » ?**

Les définitions concernant les produits et équipements requis pour les services sécurisés, moins sécurisés et les moins sécurisés ont été élaborées en collaboration avec l'équipe Prévention de l'avortement non sécurisé de l'OMS, conformément aux recommandations du Guide de soins liés à l'avortement de l'OMS.

#### **D'où proviennent les données sur les taux de complications dans l'onglet « Définitions » ?**

Il est difficile de définir de manière appropriée les taux de complications pour tous les services en établissement et hors établissement en raison du manque de données empiriques. Les sources de données pour les taux de complications inclus dans le modèle sont référencées dans les Annexes C et D, et les détails de calcul pour les estimations utilisées sont décrits dans la section méthodologie du modèle de ce document.

#### **Comment utiliser l'assistant IA ?**

Assurez-vous d'avoir ouvert l'onglet « IA » en bas de la page, s'il n'est pas déjà ouvert. Une fois l'onglet « IA » ouvert, vous pouvez utiliser l'assistant IA en tapant votre requête dans la zone de chat, puis en appuyant sur « soumettre ».

#### **Puis-je modifier les paramètres des scénarios par défaut ?**

Oui, vous pouvez ! Il suffit d'ajuster le curseur comme vous le feriez lors de l'ajustement des valeurs du scénario de référence. Vous verrez vos modifications reflétées dans la plateforme.

#### **Puis-je ajouter un nouveau scénario par défaut ?**

À ce stade, il n'est pas possible d'ajouter de nouveaux scénarios par défaut. Cependant, vous pouvez créer un scénario personnalisé pour modéliser exactement le scénario que vous souhaitez explorer, ou modifier un scénario par défaut défini pour mieux correspondre à votre question de recherche.

#### **Comment les scénarios par défaut ont-ils été décidés ?**

Les scénarios par défaut actuellement dans l'outil reflètent certains des scénarios qui pourraient avoir l'impact positif le plus significatif sur l'augmentation de l'accès aux services d'avortement sécurisé. Ceux-ci incluent l'amélioration de l'accès aux établissements et de leur préparation, ainsi que l'augmentation de la préférence pour les soins en établissement. Ces scénarios, ainsi que les scénarios par défaut basés sur les composantes des systèmes de santé, reflètent comment diverses interventions de santé publique pourraient permettre un meilleur accès aux services d'avortement sécurisé.
